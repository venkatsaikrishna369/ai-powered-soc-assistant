# 🛡️ AI Powered SOC Assistant

An AI-driven Security Operations Center (SOC) Assistant designed to help security analysts detect, analyze, and respond to network threats efficiently using machine learning and intelligent automation.

---

## 🚀 Project Overview

The **AI Powered SOC Assistant** is a cybersecurity intelligence dashboard that simulates real-world SOC workflows. It helps in analyzing network logs, detecting anomalies, and assisting analysts with threat intelligence using AI/ML techniques.

This project is built to demonstrate practical skills in:
- Cybersecurity operations (SOC workflows)
- Machine Learning for anomaly detection
- Log analysis and threat intelligence
- Python-based backend development

---

## 🎯 Key Features

- 🔍 Log ingestion and preprocessing pipeline
- 🤖 AI/ML-based anomaly detection system
- 📊 Security dashboard for visualization
- 🚨 Threat detection and alert generation
- 🧠 OpenAI-powered intelligent assistant (for SOC insights)
- 📁 Structured dataset handling (training/testing logs)
- ⚙️ Config-driven architecture for scalability

---

## 🧠 Tech Stack

- Python 
- Scikit-learn
-  ML libraries
- Pandas, NumPy
- OpenAI API (for AI assistance)
- FastAPI 
- Matplotlib (for visualization)
- Git & GitHub for version control

## ⚙️ Setup & Installation

# Clone repository

git clone https://github.com/Avenkatsaikrishna369/AI-Powered-SOC-Assistant.git

# Navigate into project
cd AI-Powered-SOC-Assistant

# Create virtual environment
python -m venv venv
venv\Scripts\activate   # Windows

# Install dependencies
pip install -r requirements.txt

Create a .env file:

OPENAI_API_KEY=your_api_key_here

▶️ How to Run
python main.py
